# 視覺組合版

更新內容：
1. 底色與線條色只從三組指定搭配中使用：
   - 紫綠組：#B020FF / #7FFF00
   - 藍黃組：#0000FF / #FFFF00
   - 青粉組：#64DCFF / #FFAA8C

2. 線寬與間距：
   - 線寬 3px
   - 間距 3px
   - 兩者一樣寬，符合需求 2~4px。

3. 圓直徑：
   - 主要設定 260px
   - 在超寬螢幕 6 張並排時自動縮為 220px，避免卡牌被切掉。
   - 仍在 200~400px 範圍內。

4. 每回合：
   - 6 個候選圓
   - 5 張普通牌 + 1 張功能牌
   - 5 張普通牌中有 1 張正確答案
   - 另外 4 張由剩下 5 個圓隨機抽，不重複

5. Google Sheet 新增欄位：
   - 視覺組合
   - 線寬px
   - 間距px
   - 圓直徑px
